將py打包成exe可執行檔

  • 首頁

安裝模組

使用cmd或gitbash輸入指令進行模組安裝
pip install pyinstaller

指令打包

使用cmd或gitbash輸入指令將main.py進行打包 *在打包前先確定路徑為.py所在的資料夾*
pyinstaller -F main.py
  • -F 打包成一個exe文件
    • 
								
  • –icon=圖標路徑
    • 
								
  • -w 使用視窗，無控制台
    • 
								
  • -c 使用控制台，無視窗
    • 
								
  • -D 創建一個目錄，包含exe以及其他一些依賴性文件
